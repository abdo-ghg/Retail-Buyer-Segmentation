About This Application

Understanding customer segmentation through machine learning

Purpose

This web application uses machine learning algorithms to segment retail customers based on their purchasing behavior, demographics, and spending patterns. It helps businesses understand their customer base better and make data-driven marketing decisions.

Technology Stack

  • Backend: Flask (Python Web Framework)
  • ML Libraries: Scikit-learn, Pandas, NumPy
  • Visualization: Matplotlib, Seaborn
  • Frontend: HTML5, CSS3, Bootstrap 5
  • JavaScript: Vanilla JS for interactivity

Features

Manual Input

Enter customer data manually for quick predictions

CSV Upload

Process bulk customer data via CSV files

K-Means Clustering

Automatic customer segmentation using ML

Visual Analytics

Interactive charts and visualizations

How It Works

  1. Data Input: Upload CSV file or enter data manually
  2. Preprocessing: Data is cleaned and normalized
  3. Feature Engineering: Calculate derived features like total spending
  4. Clustering: K-Means algorithm groups similar customers
  5. Analysis: Generate insights and visualizations
  6. Results: Display segments with actionable recommendations

Data Privacy

All uploaded data is processed locally and not stored permanently. Files are deleted after analysis. Your customer data remains private and secure.
